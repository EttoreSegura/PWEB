Mobile First



  1. Introdução

    2. O crescente uso dos dispositivos móveis para realizar tarefas do cotidiano ou para acessar a internet, Luke Wroblewski criou o conceito Mobile First em 2011 propondo ter o foco inicial do desenvolvimento de um projeto web voltado para os dispositivos móveis e posteriormente adaptar para o desktop.
    Assim, priorizando a experiência do usuário móbile.




  2. Vantagens

    3. Quando um site é inicialmente desenvolvido para desktop, a versão mobile precisa de muitas adequações nos elementos para que tudo funcione no espaço reduzido do celular. Em alguns casos, muitos componentes são removidos para que o site fique perfeito no ambiente mobile. Em uma tela maior é possível adicionar mais elementos e informações, mas na tela reduzida de um celular, o excesso de informações pode levar a uma má experiência do usuário, assim tendo que eliminar ou ocultar certos elementos.
    Dessa forma, um site desenvolvido para dispositivos móveis é mais facilmente ajustado para o desktop, já que passar a estrutura de uma tela menor para uma maior é mais simples, e proporciona uma experiência mais fluida e prática para usuários mobile.




  3. Conclusão

    4. Com o alto numero de usuários e a praticidade dos dispositivos móveis, o foco no desenvolvimento mobile se torna algo indispensável para que se possa proporcionar uma experiência agradável para os usuários.
